# HexSoftwares_Web-Music-Player
Aura Music Player – A modern dark-themed web music player built with HTML, CSS, and JavaScript featuring audio playback, personal music library management, drag-and-drop uploads, IndexedDB storage, playlist controls, and responsive glassmorphism UI. Developed as an internship project at HexSoftwares.

# 🎵 Aura Music Player

Aura Music Player is a modern web-based music player built using HTML5, CSS3, and Vanilla JavaScript. The application provides a sleek dark-themed glassmorphism interface with built-in playlists, personal music library management, drag-and-drop uploads, persistent storage using IndexedDB, and advanced playback controls.

This project was developed as part of my internship at **HexSoftwares**.

---

## 🚀 Features

### 🎶 Music Playback
- Play, Pause, Next, Previous controls
- Shuffle mode
- Repeat All / Repeat One modes
- Interactive progress bar
- Volume control with mute functionality
- Real-time playback updates

### 📂 My Library
- Upload personal audio files
- Drag & Drop support
- Persistent storage using IndexedDB
- Delete tracks from library
- Automatic track duration detection

### 🔍 Search Functionality
- Search built-in playlists
- Search uploaded music library
- Keyboard shortcut support (Ctrl + F)

### 🎨 Modern UI
- Glassmorphism design
- Dark theme interface
- Responsive layout
- Animated player controls
- Toast notifications
- Interactive modals

### ⌨️ Keyboard Shortcuts
| Shortcut | Action |
|-----------|---------|
| Space | Play / Pause |
| ← | Previous Track |
| → | Next Track |
| ↑ | Volume Up |
| ↓ | Volume Down |
| S | Shuffle |
| R | Repeat |
| M | Mute |
| Ctrl + F | Search |
